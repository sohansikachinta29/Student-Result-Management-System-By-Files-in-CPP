# Student Result Management System (C++)

## Description
A menu-driven C++ project that manages student results using:
- Structures
- File handling (ifstream, ofstream)
- Functions

The program allows users to:
1. Add student records
2. Store data in a text file
3. Read and display student records from the file

## Features
- Calculates total, average, and grade
- Handles multiple students
- Uses file-based storage
- Beginner-friendly and well-structured code

## Concepts Used
- C++ Structures
- File Handling
- Functions
- Loops and Conditions
